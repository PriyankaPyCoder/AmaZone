# 🛒 AmaZone

A responsive frontend clone of Amazon's website, built from scratch using pure **HTML & CSS** — no frameworks, no libraries.

This project was built as a hands-on way to practice real-world layout structuring, responsive design, and recreating a complex, production-grade UI using only core web fundamentals.

---

## 📌 Overview

Amazon's homepage looks simple at first glance, but it's made up of dozens of independent sections — navbar, product grids, deal banners, footer — each with its own spacing, alignment, and responsiveness rules.

**AmaZone** recreates that structure using:
- Semantic HTML for clean, accessible markup
- Flexbox for navigation and content alignment
- CSS Grid for structured, multi-column layouts
- Responsive design principles to adapt across screen sizes

---

## ✨ Features

- 🧭 Responsive navigation bar
- 🛍️ Product grid layout
- 🔥 Deals/offers section
- 📱 Mobile-friendly responsive design
- 🎨 Clean, Amazon-inspired visual styling

---

## 🛠️ Built With

- **HTML5** — structure and content
- **CSS3** — styling, Flexbox, and Grid layouts

## 💻 Getting Started

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/PriyankaPyCoder/amazone.git

# Navigate to the project folder
cd amazone

# Open index.html in your browser
```

No build tools or dependencies required — just open `index.html` directly in any browser.

## 🎯 Challenges & Learnings

Replicating Amazon's layout without a CSS framework meant manually handling spacing, alignment, and breakpoints for every section. This project strengthened my understanding of:
- Responsive layout structuring
- Flexbox and Grid working together in a single design
- Breaking a complex UI into manageable, independent components
